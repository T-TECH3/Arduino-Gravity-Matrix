## Description
The Arduino Gravity Matrix is an embedded systems project that simulates real-time gravitational physics on a physical dot matrix display. The system utilises an Arduino Uno microcontroller paired with an Inertial Measurement Unit (IMU) and a 8x8 Led Matrix. The code accounts for the IMU drift and dynamically calculates the state of the LEDs.
